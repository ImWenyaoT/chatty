# 西装商品说明

西装类商品适合面试、婚礼、主持、商务活动等场景。

客服推荐时应优先确认：
1. 使用场景
2. 身高体重
3. 使用日期
4. 是否需要整套搭配

## SUIT-001 尺码参考

商品编号：SUIT-001

租赁价格口径：
1. 第一天租赁价格 199 元
2. 续租每天 99.5 元
3. 在途时间不计入租期
4. 寄出一般包邮，新疆、西藏等偏远地区除外

尺码建议示例：
1. 身高 168cm - 174cm，体重 55kg - 65kg，优先建议 M
2. 身高 175cm - 181cm，体重 66kg - 80kg，优先建议 L
3. 身高 182cm - 188cm，体重 81kg - 92kg，优先建议 XL

如果客户只提供了身高体重，没有肩宽、胸围、腰围，客服可以先给出上述初步建议，并说明最终以试穿或详细尺码数据为准。

客服引导建议：
1. 如果用户先问“怎么租”，先说明第一天全价、续租半价、在途不算租期。
2. 然后继续收集使用日期和身高体重。
3. 如果尺码和档期都满足，可以直接引导用户下单，并填写使用日期对应的租赁时间。
