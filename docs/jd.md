## 团队使命
**Model + Harness = Agent**
以研究、工程、产品相结合的方式，把 llm 的模型能力转化为前沿的科研突破与领先的 Agent 产品。加入 Harness 团队，与研究员、工程师、产品经理紧密协作，探索 Harness 领域的研究、工程与产品前沿，定义 llm 对 Harness 的理解。

---

## 方向一 · 技术方向
探索 Agent Harness 方向的研究前沿。
探索并解决面向下一代 Agent Harness 的工程难题。

### 主要职责
1. 参与设计 llm Harness 产品的技术架构与选型。- CS 及其它学术领域含金量论文发表。
2. 参与开发 llm Harness 产品。
3. 与研究员、工程师深度协作、紧密合作，共同定义并实现 Harness 领域基于模型能力的前沿创新，包括但不限于：上下文管理、长期记忆、Subagent 与 Multi-Agent、自进化 Agent、超长程任务等。
4. 与模型训练团队深度协作，从 Harness 角度实现 llm 的 Harness 与模型深度适配，参与实现模型与 Harness 持续共同进化。
5. 提出 Harness 领域的基准测试与评测方法，构建评测基准数据、制定数据标注策略，优化 Agent 在各领域的智能水平。
6. 以真实世界任务作为 Harness 研究的重要反馈源，设计相关数据与实验，持续迭代 Agent 在真实场景下的表现。
7. 以内部真实任务作为 Harness 产品和模型能力训练的重要反馈源，持续迭代产品能力。
8. 理解并分析用户反馈；在有兴趣且力所能及前提下，协助维护 Harness 产品用户社群。
9. 协助项目管理相关工作。

### 任职要求
1. 计算机科学或相关领域科研经验，技术水平过硬、技术眼界广阔、有科研品味。
3. 计算机科学领域具有**含金量的科研成果**。
4. 能独立分析问题并提出 idea，具备从 0 到 1 推动研究的能力，能快速将想法转化为可运行原型，高效迭代实验。
5. 熟练用 AI Agent 工具做开发；在软件开发领域有极强学习能力；能在 AI 辅助下能于无直接经验的领域（语言 / 技术 / 框架）做**有质量保证**的研究目的的编程。
6. Agent 产品的**高强度用户**，对 Agent Harness 开发有极大热情，对模型行为有品味有判断力，**对开发者体验有强感知**；深度使用代码类及通用类 Agent 产品并融入工作生活。
7. 熟悉 LLM 与 Agent 基本机制：LLM API、KV Cache、Agent Loop、Tool Use、Reasoning、Planning、Skills、MCP、Memory、Subagent、Multi-Agent；对 Prompt / Context / Harness Engineering 有**较深入理解**。
8. 良好的中文沟通能力。

### 加分项
- AI 领域科研经验、研究员任职经验。
- 个人开源作品及开源社区深度贡献。
- CS 及其它学术领域含金量论文发表。
- CS / 编程 / 数学 / 数据科学含金量比赛奖项。
- 其它超乎常人的相关才能。
- AI 行业相关软件开发经验。
- 作为小团队研发工程师**快速迭代产品**的经验。
- 与研究员深度协作的经验或科研经验，有科研品味。
- 能用英文与开源社区、用户社群沟通。

## 方向二 · 产品方向
探索并定义下一代 Agent 产品形态。

### 主要职责
1. 规划 llm Harness 产品路线图，连接研究员、工程师、开源社区和广大用户。
2. 理解判断用户最真实的需求，定义与衡量"Agent 是否真的在更多场景下更深入地帮助到更多的人"的指标。
3. 帮助 Harness 产品落地，持续跟踪真实场景下的用户任务，推动产品创新和体验优化。
4. 维护 Harness 产品用户社群，从潜在海量用户群体中获取反馈、提取信号，指导产品迭代。
5. 与模型训练团队研究员深度协作，实现模型与 Harness 共同进化。
6. 协助项目管理相关工作。

### 任职要求
1. 产品逻辑清晰。
2. 能够使用 **vibe coding** 写代码（不一定需要技术背景）。
3. 具备 **UI/UX 设计素养**，能在 AI 辅助下完成产品原型图设计、UI 设计等工作。
4. Agent 产品的高强度用户，熟悉各种产品形态，对开发者体验及用户体验有强感知；深度使用过 Claude Code、Cowork、Codex、Cursor、OpenCode、GitHub Copilot、Manus、OpenClaw、Hermes 等并融入工作生活。
5. 理解 LLM 与 Agent 基本机制：LLM API、KV Cache、Agent Loop、Tool Use、Reasoning、Planning、Skills、MCP、Memory、Subagent、Multi-Agent；对 Prompt / Context / Harness Engineering 有**第一手实践**。
6. 能设计**系统性的数据收集方法**（问卷、访谈、A/B 测试、灰度测试等），并用统计学工具严谨科学地分析数据，对此有系统性思考和实践。
7. 对产品细节有强感知力，对异常分支、边界条件和失败场景有敏锐的嗅觉。
8. 优秀的中文沟通能力；能用英文与开源社区、用户社群书面沟通。

### 加分项
- AI 行业从业经验、AI 相关产品经验。
- 作为小团队产品经理主导产品路线的经验。
- 与研究员深度协作的经验。
- 深度参与开源社区或维护开源社区关系的经验。
- 其它超乎常人的相关才能。
